# 완전초보자를 위한 회계자료등록 마법사

## Executive Summary

| 관점 | 내용 |
|------|------|
| **Problem** | 정치자금 회계에 익숙하지 않은 초보 회계책임자가 계정/과목/지출유형 등 복잡한 코드 체계를 이해하지 못해 자료 등록에 어려움을 겪음 |
| **Solution** | 단계별 마법사(Wizard) UI로 자연어 질문에 답하면 시스템이 적합한 코드값을 자동 선택하여 등록을 완료하는 가이드 방식 |
| **Function UX Effect** | "어떤 종류의 거래인가요?" → "누구에게?" → "얼마?" 3단계 질문으로 복잡한 회계 입력을 단순화, 초보자도 1분 이내 등록 가능 |
| **Core Value** | 회계 전문지식 없이도 정확한 정치자금 장부 입력이 가능해져 선거 회계 진입 장벽을 대폭 낮춤 |

## 1. 배경 및 문제

### 현재 상황
- 수입내역관리/지출내역관리 페이지는 **계정 → 과목 → (지출유형 3단계) → 거래처 → 금액** 순서로 입력
- 계정(후보자등자산, 후원회기부금, 보조금, 보조금외지원금)과 과목(선거비용, 선거비용외) 개념을 모르면 입력 불가
- 지출유형은 11개 대분류 × 수십 개 중분류 × 소분류로 선택지가 너무 많음
- 초보 회계책임자는 "현수막 제작비는 어디에 입력해야 하나요?" 같은 기본 질문에서 막힘

### 대상 사용자
- 처음 선거 출마하는 후보자의 회계책임자
- 정당/후원회의 신규 담당자
- 회계 프로그램 사용 경험이 없는 자원봉사자

## 2. 목표

1. **초보자도 3단계 이내에 자료 등록 완료** (계정/과목/지출유형 직접 선택 없이)
2. **자연어 기반 거래 유형 분류** ("현수막 만들었어요" → 선거비용 > 거리게시용현수막)
3. **실수 방지**: 잘못된 코드 조합이 불가능하도록 마법사가 유효한 값만 제시
4. **기존 시스템과 100% 호환**: 마법사를 통해 등록한 데이터는 기존 수입/지출 페이지에서 동일하게 조회·수정 가능

## 3. 핵심 기능

### 3.1 마법사 흐름 (지출)

```text
Step 1: 거래 유형 선택 (시각적 카드)
┌──────────────┐ ┌──────────────┐ ┌──────────────┐
│  🏢 사무소    │ │  📄 인쇄물    │ │  📢 광고/홍보  │
│  임대료,관리비 │ │  명함,공보물   │ │  신문,TV,인터넷 │
└──────────────┘ └──────────────┘ └──────────────┘
┌──────────────┐ ┌──────────────┐ ┌──────────────┐
│  🎽 소품      │ │  🚗 차량/이동  │ │  📱 전화/문자  │
│  어깨띠,모자   │ │  임차,유류비   │ │  통화,SMS발송  │
└──────────────┘ └──────────────┘ └──────────────┘
┌──────────────┐ ┌──────────────┐ ┌──────────────┐
│  👥 인건비    │ │  🏗 현수막/설치 │ │  📦 기타      │
│  사무원수당    │ │  현수막,무대   │ │  그 외 지출    │
└──────────────┘ └──────────────┘ └──────────────┘

→ 시스템이 자동으로: 계정 + 과목 + 지출유형1 + 지출유형2 설정

Step 2: 세부 정보 입력 (간소화된 폼)
- 거래처명 (+ 자동완성)
- 금액
- 날짜 (기본값: 오늘)
- 내역 설명 (자유 입력)
- 결제수단 (카드/현금/계좌이체)

Step 3: 확인 및 저장
- 자동 설정된 계정/과목/지출유형 표시 (수정 가능)
- "이 내용으로 등록하시겠습니까?" 확인
- 증빙서번호 자동 채번
- 증빙파일 첨부 (선택)
```

### 3.2 마법사 흐름 (수입)

```text
Step 1: 수입 유형 선택 (시각적 카드)
┌──────────────┐ ┌──────────────┐ ┌──────────────┐
│  💰 후원금    │ │  🏛 보조금    │ │  🏦 자산      │
│  기명/익명    │ │  정당 지원금   │ │  후보자 자산   │
└──────────────┘ └──────────────┘ └──────────────┘
┌──────────────┐
│  📦 기타수입  │
│  이자,기타    │
└──────────────┘

→ 시스템이 자동으로: 계정 + 과목 설정

Step 2: 세부 정보 (간소화된 폼)
Step 3: 확인 및 저장
```

### 3.3 거래 유형 → 코드 매핑 테이블

| 카드 | 계정 | 과목 | 지출유형1 | 비고 |
|------|------|------|----------|------|
| 사무소 | (기관별 첫 계정) | 선거비용외 | 선거사무소 | 임대/관리비/비품 |
| 인쇄물 | (기관별 첫 계정) | 선거비용 | 인쇄물 | 명함/공보/홍보물 |
| 광고/홍보 | (기관별 첫 계정) | 선거비용 | 광고 | 신문/TV/인터넷 |
| 소품 | (기관별 첫 계정) | 선거비용 | 소품 | 어깨띠/모자/윗옷 |
| 차량/이동 | (기관별 첫 계정) | 선거비용 | 공개장소연설대담 | 차량 중분류 |
| 전화/문자 | (기관별 첫 계정) | 선거비용 | 전화/전자우편/문자메시지 | |
| 인건비 | (기관별 첫 계정) | 선거비용 | 선거사무관계자 | |
| 현수막/설치 | (기관별 첫 계정) | 선거비용 | 거리게시용현수막 | |
| 기타 | (기관별 첫 계정) | 선거비용 | 기타 | 사용자가 지출유형 직접 선택 |
| 후원금 | 후원회기부금 | (첫 과목) | - | 기명/익명 구분 |
| 보조금 | 보조금 | (첫 과목) | - | |
| 자산 | 후보자등자산 | (첫 과목) | - | |

### 3.4 자연어 검색 보조

"기타" 카드를 선택하거나, 어떤 카드를 골라야 할지 모를 때:
- 텍스트 입력란에 "현수막", "사무실 전기세", "명함 인쇄" 등 키워드 입력
- 키워드 매핑으로 적합한 카드/지출유형 하이라이트 추천

## 4. 구현 범위

### 구현 대상
- `/dashboard/wizard/page.tsx` — 마법사 메인 페이지
- 거래 유형 카드 컴포넌트
- 3단계 폼 (Step 1 → 2 → 3 진행 바)
- 코드 매핑 로직 (`lib/wizard-mappings.ts`)
- 사이드바 메뉴 추가 ("간편등록 마법사")

### 재사용 코드
- `useCodeValues()` — 계정/과목 조회
- `CustomerSearchDialog` — 거래처 검색/등록
- `/api/acc-book` action: "insert" — 최종 저장
- `/api/acc-book` maxRcpNo — 증빙서번호 채번
- `/api/evidence-file` — 증빙파일 첨부
- `lib/expense-types.ts` — 지출유형 데이터

### 구현 제외
- AI 기반 자연어 분류 (키워드 매핑으로 충분)
- 일괄등록 (기존 batch-import 사용)
- 기존 수입/지출 페이지 변경 (마법사는 독립 페이지)

## 5. 기술 설계 방향

### UI 구조
```text
WizardPage
├── StepIndicator (1/3, 2/3, 3/3 진행 바)
├── Step1: TypeSelector (카드 그리드 + 키워드 검색)
├── Step2: DetailForm (거래처, 금액, 날짜, 내역, 결제수단)
├── Step3: ConfirmSave (자동설정값 표시 + 수정 가능 + 저장)
└── CustomerSearchDialog (거래처 검색)
```

### 상태 관리
- `useState`로 step(1~3), selectedType, formData 관리
- 이전 단계로 돌아가기(뒤로) 지원
- Step 3에서 계정/과목 수정 시 CodeSelect 재사용

### 데이터 흐름
```text
사용자 카드 선택 → lib/wizard-mappings.ts에서 코드 조회
  → useCodeValues()로 유효성 확인 (acc_rel 기반)
  → Step 2 폼 입력 → Step 3 확인
  → /api/acc-book insert + /api/evidence-file (선택)
```

## 6. 성공 지표

- 초보자가 첫 자료 등록까지 **1분 이내**
- 계정/과목 선택 오류 **0건** (마법사가 유효 조합만 제시)
- 기존 고급 사용자는 영향 없음 (마법사는 별도 메뉴)
