## 1. 배경 및 필요성

### 1.1 부담비용이란?

선거운동 관련 정당/후보자를 위해 선관위가 업무를 수행하고, 그 비용을 국가/지자체가 직접 부담하는 것. **선거비용외 정치자금** 과목에서 지출 처리함.

### 1.2 부담비용 대상 항목

| 항목 | 세부 | 비고 |
|------|------|------|
| **활동보조인** | 수당, 실비, 산재보험료 | 고용보험료는 제외 (선거비용외로 별도 처리) |
| **점자형 선거공보** | 지대, 인쇄비, 제본비, 한글 1도 인쇄료, 운반비 | 기획/도안료는 미지급 |
| **점자형 선거공약서** | 지대, 인쇄비, 제본비 | 발송용 봉투/우편비는 보전대상 선거비용 |
| **저장매체** | 제작비, 디지털파일 전환비, 운반비 | 시각장애선거인용 |

### 1.3 현재 시스템 Gap

- `reimbursement/page.tsx`: 선거비용 보전 대상 체크만 지원 (선거비용외 제외)
- `expense-types.ts`: 부담비용 관련 지출유형은 존재하나 부담비용 여부 판별 로직 없음
- 서식7(부담비용 지급청구서) 생성 기능 없음
- 부담비용 해당 항목 자동 필터링 없음

## 2. 기능 요구사항

### 2.1 부담비용 지출내역 조회 (FR-01)

- 선거비용외 정치자금 중 부담비용 해당 지출만 필터링하여 조회
- 부담비용 판별 기준:
  - 과목: 선거비용외 정치자금
  - 지출유형1(expGroup1) 기준:
    - `"선거사무관계자"` + 지출유형2 `"선거사무관계자수당"` + 지출유형3 `"활동보조인"` → 수당/실비/산재보험료만
    - 별도 부담비용 카테고리로 점자형 선거공보, 저장매체, 점자형 선거공약서 관련 지출
- 기간 필터, 항목별 필터 제공

### 2.2 부담비용 체크 및 금액 집계 (FR-02)

- 각 지출건에 부담비용 해당 여부 체크박스
- 부담비용 총액 자동 합산
- 항목별(점자형 선거공보, 점자형 선거공약서, 저장매체, 활동보조인) 소계 표시

### 2.3 부담비용 지급청구서 생성 (FR-03)

- 서식7 양식에 맞는 Excel 또는 PDF 출력
- 자동 입력 항목:
  - 선거명, 소속정당명, 후보자명 (organ 테이블에서)
  - 점자형 선거공보 등 작성/제출 수량 (사용자 입력)
  - 청구금액 (체크된 지출 합산, 항목별 분류)
  - 수령계좌 정보 (사용자 입력)
- 첨부서류 안내 체크리스트

### 2.4 기존 보전 페이지 통합 (FR-04)

- `reimbursement/page.tsx`에 탭 추가: `선거비용 보전` | `부담비용 청구`
- 선거비용 보전 탭: 기존 기능 유지 (선거비용만)
- 부담비용 청구 탭: 선거비용외 부담비용 조회/체크/청구서 생성

### 2.5 부담비용 판별 로직 (FR-05)

- `expense-types.ts`에 부담비용 판별 함수 추가
- 지출유형 선택 시 "이 항목은 부담비용에 해당합니다" 안내 표시
- wizard에서 관련 지출 등록 시 부담비용 안내 제공

## 3. 비즈니스 규칙

### 3.1 청구 관련

| 항목 | 규칙 |
|------|------|
| 청구권자 | 점자형 선거공보 등을 제출/발송하거나 활동보조인을 선임한 모든 정당 또는 후보자 |
| 청구기한 | 선거일 후 10일 (2026.6.15까지) |
| 지급기한 | 2026.7.31까지 |
| 지급방법 | 청구서에 기재된 예금계좌에 입금 |
| 필수 첨부 | 수입/지출부 사본, 활동보조인 수당/실비 지급명세서, 영수증 사본, 통장 사본 |

### 3.2 부담 기준 세부

- 활동보조인 식사 제공 시 → 실비에서 해당 금액 감함
- 점자형 선거공보/공약서는 통상적 인쇄가격 범위 내에서만
- 한글 게재사항 외 문자/그림/사진 인쇄비, 기획/도안료는 미지급
- 활동보조인 고용보험료는 부담비용 아님 (선거비용외로 별도 처리)
- 예비후보자의 활동보조인 포함, 단 후보자등록 미완 시 미부담

### 3.3 보전비용 인계 시 공제

- 부담비용(F)은 보전비용 인계 계산에서 공제 대상
  - F = 후보자 자산계정의 점자형 선거공보, 저장매체, 점자형 선거공약서, 활동보조인 수당/실비 등에 대해 선관위로부터 받은 부담비용
